# 🎮 3D Horror Game (Unity)

## 📌 Description
This project is a basic 3D horror game developed using Unity.  
The game focuses on creating an immersive horror environment through lighting effects, sound design, and character interaction mechanics.

The project demonstrates fundamental skills in 3D game development, gameplay logic, and environment design.

---

## 🚀 Features

- 🎮 Player movement and interaction system
- 🌫️ Horror environment with lighting and visual effects
- 🔊 Sound effects and background audio for immersive gameplay
- 🧍 Character animation integration
- 🧠 Basic gameplay logic and event handling
- 🗺️ 3D environment and scene design

---

## 🛠️ Technologies Used

- **Unity Game Engine**
- **C# Programming**
- **Unity Animation System / Mixamo**
- **Audio Editing Tools (Audacity / Adobe Audition)**
- **Lighting & Visual Effects (Fog, Shadow, Bloom)**
- **GitHub (Version Control)**

---

## 🎯 Project Purpose

- Practice 3D game development using Unity
- Implement player interaction and game mechanics
- Apply lighting and sound techniques for horror atmosphere
- Gain experience in scene design and asset management
